# 🛡️ DungeonKeeper Bot Overview

**DungeonKeeper** is a multi-purpose Discord bot built for community management, moderation, and member engagement. It specialized in high-efficiency **ModMail** and **Staff Automation**.

## 🚀 Core Systems

### 📬 ModMail System
The heart of DungeonKeeper. It allows users to communicate with server staff privately through the bot's DMs.
- **Privacy**: User messages are forwarded to a dedicated staff channel/thread.
- **Safety**: Users can be blacklisted if they abuse the system.
- **Clean UI**: Uses interactive buttons and clean text formatting for easy management.

### ⚒️ Staff Automation
Automates the administrative overhead of managing a staff team.
- **Promotion/Demotion Logs**: Instantly notifies staff members of their status changes with branded embeds.
- **Action Logs**: Every major command (warn, mute, ban, DM) is logged in a secure channel for accountability.

### 💎 Premium Membership
Unlocks enhanced interactions for dedicated community members.
- **Action Commands**: Access to exclusive interactive GIF commands like `-hug`, `-slap`, and `-punch`.
- **Advanced Matchmaking**: Access to the `-ship` command with high-fidelity image generation.

## 🛠️ Command Categories
- **ModMail Management**: `reply`, `close`, `blacklist` (Interactive Buttons)
- **Staff Tools**: `/dm`, `/say`, `/announce`, `/rules`, `/dm_warning`
- **User Management**: `/blacklist_add`, `/blacklist_remove`, `/modmail_status`
- **Premium Actions**: `-hug`, `-kick`, `-slap`, `-kiss`, `-punch`, `-ship`
